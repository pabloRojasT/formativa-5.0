# Sitio Web de Verduras

Este es un sitio web simple para una tienda de verduras frescas. Incluye páginas para inicio, acerca de, productos y contacto.

## Estructura del Proyecto

- `index.html`: Página de inicio
- `about.html`: Página acerca de nosotros
- `products.html`: Página de productos
- `contact.html`: Página de contacto
- `style.css`: Estilos CSS para el sitio

## Cómo Usar

1. Abre `index.html` en tu navegador web para ver el sitio.
2. Navega entre las páginas usando el menú de navegación.

## Características

- Diseño responsivo básico
- Formulario de contacto simple
- Lista de productos con precios

## Notas

- El formulario de contacto no está conectado a un backend; es solo para demostración.
- Para producción, considera agregar imágenes y un servidor backend.
